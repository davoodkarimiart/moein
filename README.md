# 📸 اتلیه معین - Photography Studio Management System

وبسایت حرفه‌ای استودیوی عکاسی "اتلیه معین" با پنل مدیریتی برای ادمین و کارمندان.

## ✨ ویژگی‌ها

### 🌐 صفحات عمومی
- **صفحه اصلی**: هیرو سکشن، خدمات، نمونه کارها، نظرات مشتریان
- **درباره ما**: تاریخچه، تیم، ارزش‌های اتلیه
- **خدمات**: ۶ خدمت عکاسی مختلف + بسته‌های قیمتی
- **گالری**: نمایش نمونه کارها
- **تماس**: فرم تماس، اطلاعات تماس، نقشه

### 👤 سیستم احراز هویت
- ورود برای ادمین و کارمندان
- ذخیره‌سازی localStorage برای جلسات
- **حساب demo**: 
  - ایمیل: `admin@moein.com`
  - رمز: `123456`

### 👨‍💼 داشبورد مدیر (Admin)
- **نمای کلی**: آمار کل کارمندان، عکس‌ها و رزروها
- **مدیریت کارمندان**: افزودن، ویرایش، حذف کارمندان
- **مدیریت گالری**: آپلود و حذف عکس‌های نمونه‌کار
- **سایدبار**: ناوبری راحت و سریع

### 👨‍💼 داشبورد کارمند (Employee)
- **نمای کلی**: آمار شخصی
- **گالری شخصی**: نمایش عکس‌های شخصی
- **ثبت رزرو**: فرم برای ثبت رزروهای جدید

## 🛠️ فناوری‌های استفاده‌شده

```
Frontend:
- React 18+
- React Router v6 (ناوبری)
- Tailwind CSS v4 (طراحی)
- Lucide React (آیکون‌ها)
- Vite (build tool)

Backend State:
- Context API (احراز هویت و داده‌ها)
- localStorage (ذخیره‌سازی)

Styling:
- RTL Support برای فارسی
- Responsive Design (موبایل تا دسکتاپ)
- فونت Vazirmatn
```

## 📁 ساختار پروژه

```
src/
├── components/
│   ├── layout/          (Header, Footer, Sidebar)
│   ├── common/          (Button, Card, Modal)
│   └── sections/        (HeroSection, ServiceCard, etc)
├── pages/               (تمام صفحات)
├── context/             (AuthContext)
├── App.jsx
├── main.jsx
└── index.css           (Tailwind + Custom CSS)
```

## 🚀 نصب و اجرا

### نیازمندی‌ها
- Node.js 16+
- npm یا yarn

### راه‌اندازی

```bash
# نصب وابستگی‌ها
npm install

# اجرای development server
npm run dev

# بسته‌سازی برای production
npm run build

# پیش‌نمایش build
npm run preview
```

سرور در `http://localhost:3000` اجرا می‌شود.
